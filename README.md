# COVID-19 Data Analysis Project

## 📌 Project Overview
This project performs exploratory data analysis (EDA) on global COVID-19 datasets using Python. The project analyzes country-wise COVID statistics, recovery rates, death rates, weekly growth, and WHO region analysis while generating meaningful visualizations and dashboards.

---

## 🚀 Features

- Global COVID-19 statistics analysis
- Country-wise confirmed cases analysis
- Death and recovery rate calculations
- Weekly growth analysis
- WHO region-wise comparison
- Top affected countries identification
- Data visualization using graphs and charts
- Dashboard generation for insights

---

## 📊 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📁 Dataset Used

Dataset:

- `country_wise_latest.csv`

Contains:

- Country / Region
- Confirmed Cases
- Deaths
- Recoveries
- Active Cases
- New Cases
- Weekly Growth Data
- WHO Regions

## 🔍 Analysis Performed

### Global Analysis

- Total confirmed cases worldwide
- Total deaths worldwide
- Total recoveries worldwide

### Country Level Analysis

- Country with highest confirmed cases

- Country with highest recoveries

### Daily Trend Analysis

- Highest new cases
- Highest new deaths
- New recoveries analysis

### Recovery & Fatality Analysis

- Death rate calculation
- Recovery rate calculation
- Countries with high fatality rates

### Weekly Growth Analysis

- Weekly case growth
- Percentage increase analysis
- Current vs previous week comparison

### WHO Region Analysis

- Region-wise confirmed cases
- Region-wise deaths
- Active cases comparison
